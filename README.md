
 
# Time_manager 2024秋Python大作业

## 使用方法  
1. 运行 `main.py` 即可启动程序。  
2. 根据运行环境的不同，相对路径可能会出现问题。需要注意以下文件中的路径：  
   - **`GUI.py`**:  
     - **Line 62 Line191**: 读取钟表图片、载入音乐
   - **`history_logger.py`**:  
     - **Line 5**: 读取历史记录  
   - **`music_player.py`**:  
     - **Line 8**: 读取音乐文件
3. 为了方便演示，番茄钟的单位实际上为秒而非分钟，可以在**`GUI.py`** **Line 97** 处修改

### 库依赖  
本工程需要依赖库 **Pillow** **pygame**
